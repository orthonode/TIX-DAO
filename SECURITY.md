# Security Policy

To report a security vulnerability in TIX-DAO, **do not open a public GitHub issue**.

Please report privately — see the full disclosure process, threat model, and scope in [`docs/SECURITY.md`](./docs/SECURITY.md).

**Response commitment:** acknowledgement within 48 hours, status update within 7 days.
